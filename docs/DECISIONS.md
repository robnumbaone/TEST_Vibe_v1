# Decisioni Architetturali

> Registra le scelte importanti e il motivo dietro

## Format

```
### [Data] - [Titolo Decisione]

**Decision**: Cosa hai scelto e perché

**Alternatives Considered**: 
- Option A (pro: X, contro: Y)
- Option B (pro: X, contro: Y)

**Chosen**: Option A perché Z

**Status**: [Active | Superseded | Deprecated]
```
